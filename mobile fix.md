# FutureSpace Mobile UX & Visual Refinement Specification

## Objective

Transform the current mobile experience from visually dense and repetitive into a premium luxury interior design experience that feels spacious, elegant, and editorial.

Target feeling:

- Luxury
- Minimal
- Architectural
- Calm
- Premium
- High-end residential

Reference brands:

- Studio MK27
- Olson Kundig
- Yabu Pushelberg
- Kelly Wearstler
- Foster + Partners

---

# 2. Improve Visual Hierarchy

## Current Problem

Everything competes for attention.

Headings, contact information, cards, labels and forms have similar visual weight.

## New Hierarchy

Level 1:
Large Editorial Heading

Level 2:
Short supporting paragraph

Level 3:
Contact Information

Level 4:
Form Fields

Level 5:
Legal/Footer

Use spacing rather than borders to separate content.

# 3. Remove Unnecessary Information Cards

Remove:

Consultation

# 4. Increase White Space

## Global Rules

Mobile section spacing:

Top padding: 80px

Bottom padding: 80px

Content gap: 32px

Element gap: 24px

Avoid stacking content tightly.

Luxury design requires breathing room.

# 11. Mobile Performance Optimization

Convert all project images to WebP.

Enable lazy loading.

Preload hero image.

Use responsive image sizes.

Compress assets below 300KB where possible.

Remove unused images from repository.

Use next-gen image formats.

Target:

LCP < 2.5s

CLS < 0.1

Mobile Lighthouse Score > 90
